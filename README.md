# Earth 3D Dashboard

[![CI](https://github.com/Michael-Blake-Donaldson/Earth-3D-Dashboard/actions/workflows/ci.yml/badge.svg)](https://github.com/Michael-Blake-Donaldson/Earth-3D-Dashboard/actions/workflows/ci.yml)

**[Live Demo →](https://michael-blake-donaldson.github.io/Earth-3D-Dashboard/)**

An interactive 3D Earth visualization and science content hub. Explore real planetary data — climate emissions, earthquake sites, coral reef health, aurora zones, forest carbon stores, and renewable energy hubs — rendered directly on a rotatable 3D globe. Each data layer links to a curated Earth science article for deeper context.

> Replace `src/assets/og-preview.png` with an actual screenshot of the globe once deployed for best social share previews.

---

## Features

### 3D Globe Engine

- Powered by **Babylon.js 7** — full WebGL rendering in the browser, no plugins
- Three mesh modes: **Globe** (sphere), **Geodesic** (icosphere), **Faceted** (low-poly)
- Three map skins: **Realistic** (NASA day texture), **Color Map**, **Night Lights**
- Procedural **atmosphere shell**, **glow pipeline**, and **auto-spin** effects
- Particle assembly intro animation on first load

### Scientific Data Layers

Six toggleable educational overlays, each with provenance metadata (source, year, last updated):

| Layer          | What you see                                  |
| -------------- | --------------------------------------------- |
| Climate CO₂    | Top 10 national emitters sized by MtCO₂/yr    |
| Seismic Quakes | 8 historic magnitude 8.6+ events              |
| Coral Reefs    | 8 major reef systems with health %            |
| Aurora Belt    | Polar auroral zones at 65–75° latitude        |
| Forest Carbon  | 6 critical carbon sinks in GtCO₂              |
| Renewable Hubs | 8 global renewable energy installations in GW |

### Story Presets

One-click configurations that simultaneously activate related layers, set the optimal mesh and skin, and open the matching article:

- **Climate Risk** — CO₂ + Renewable Hubs on Climate Heatmap skin
- **Tectonic Hazard** — Seismic Quakes on Geodesic mesh with Glow
- **Biosphere** — Forest Carbon + Coral Reefs on Realistic skin

### Content Hub

- Searchable blog listing with category filter chips
- Article detail pages with structured content, citations, and curated images
- Direct globe→article deep links from the data insight panel

### Polish & Engineering

- Onboarding tour (3-step) shown on first visit, skippable
- Layer opacity slider (20–100%)
- Three quality modes: High / Balanced / Performance
- 2D fallback panel with static map if WebGL is unavailable
- localStorage analytics ring buffer (150 events) — no external tracking
- Absolute URL resolver — navigation works correctly from any route context, including GitHub Pages subdirectory paths
- Accessibility: semantic HTML, ARIA live regions, keyboard focus states throughout

## Architecture

```
src/                        # Static site root (served as web root)
  index.html/js/css         # Globe home page
  blog.html/js/css          # Article listing
  article.html/js/css       # Article detail
  base.css                  # Design tokens & shared components
  data/
    articles.js             # All article content (centralized)
    layers/
      generatedLayers.js    # Runtime module — generated by data pipeline
  utils/
    sanitize.js             # XSS-safe text/query helpers
    articles.js             # Category filter utility
  assets/
    maps/                   # Globe texture maps (NASA day/night, normal, population)
    articleImages/          # Article hero images

data/
  raw/layers.json           # Authoritative raw layer dataset
  processed/                # Normalized output (generated)

scripts/data/
  validate-layers.mjs       # Schema validation
  build-layers.mjs          # Normalize + emit generatedLayers.js

.github/workflows/ci.yml    # CI: lint → format → test → deploy to GitHub Pages
tests/                      # Vitest unit tests
```

## Local Development

```bash
npm install
npm start          # lite-server at localhost:3002
```

## Quality Commands

```bash
npm run lint           # ESLint
npm run format:check   # Prettier
npm run test           # Vitest (11 specs)
npm run verify         # lint + format:check + test
```

## Data Pipeline

```bash
npm run data:validate  # Schema-check data/raw/layers.json
npm run data:build     # Emit processed JSON + generatedLayers.js
npm run data:refresh   # validate + build
```
